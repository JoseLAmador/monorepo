## Description

Use it when you try to show....:

```jsx
<Alert message="Hello everyone!" />
```

## Props

Use the `message` prop to send a custom message.

* Make sure to include `role="alert"` on `<Alert>` tags for accessibility.


## Usage

```jsx
import { Alert, Button } from '@asjosel/core'

<Alert />

<Button>hello world</Button>

```
